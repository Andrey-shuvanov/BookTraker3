# Training Planner

**Автор:** Шуванов Андрей

**Описание:**  
Приложение для планирования тренировок с графическим интерфейсом. Позволяет добавлять тренировки (дата, тип, длительность), фильтровать их по типу и дате, а также сохранять/загружать данные в JSON.

## Возможности
- Добавление тренировок
- Фильтрация по типу и дате
- Валидация ввода (дата в формате ГГГГ-ММ-ДД, длительность > 0)
- Автосохранение в data.json
- Сброс фильтров

## Установка и запуск
1. Убедитесь, что установлен Python 3.6+
2. Запустите:
```bash
python training_planner.py
